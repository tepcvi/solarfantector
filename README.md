# FanTector - Solar-Powered Mini Fan

This is a responsive landing page for the FanTector Solar-Powered Mini Fan product, featuring a modern futuristic design with cyan/blue and purple accents on a dark theme.

## File Structure

- `index.html` - The main HTML file
- `styles.css` - External CSS stylesheet
- `images/team/` - Directory for team member photos

## Customization

### Changing Team Member Information

You can easily update the team member information in the `index.html` file:

1. Find the "Our Team" section (around line 175)
2. Replace the placeholder names, roles, and descriptions with your actual team information
3. Update social media links as needed

### Color Scheme

The color scheme is defined in the `styles.css` file using CSS variables:

```css
:root {
    /* Primary palette - futuristic cyan/blue */
    --primary: #00E5FF;
    --primary-light: #6EFFFF;
    --primary-dark: #00B2CC;
    
    /* Secondary palette - electric purple */
    --secondary: #7B1FA2;
    --secondary-light: #AE52D4;
    --secondary-dark: #4A0072;
    
    /* Neutral colors */
    --black: #0A0E17;
    --dark-gray: #1A1F2C;
    --dark-gray-light: #2C3347;
    --white: #ffffff;
    --light-gray: #F0F5FF;
    
    /* Accent colors */
    --accent: #FF3D71;
    --accent-light: #FF8AA4;
    --accent-dark: #C60055;
}
```

You can modify these values to change the color scheme throughout the site.
