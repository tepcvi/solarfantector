Instructions

  1. Prepare square images of your team members (recommended size: 300x300 pixels)
  2. Name the files exactly as shown above: hacker.jpg, hipster.jpg, and hustler.jpg
  3. Place these files in the images/team/ directory
  4. The images will automatically appear on your website

Tips for Good Team Photos

  • Use professional headshots with good lighting
  • Ensure the face is clearly visible and centered
  • Use a neutral background if possible
  • Square aspect ratio works best with the circular crop

Social Media Links

Each team member has the following social media links:

  • Facebook
  • Instagram
  • Twitter
  • GitHub

Update the links in the HTML file to point to the actual social media profiles of your team members.
